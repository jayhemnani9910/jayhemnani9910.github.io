Featured Projects

Stock Market Data Platform

Problem: Financial analysts face fragmented data, high latency, and an inability to correlate market movements with real-time news events.
Solution: Architected a containerized, microservices-based platform featuring a hybrid ETL/ELT pipeline. The system ingests real-time stock ticks via Kafka and batch-processes news sentiment via Airflow, unifying them in a TimescaleDB data warehouse with a star schema design.

  • ▶ Processed over **1 million data points** daily with 99.9% uptime.
  • ▶ Reduced analytical query latency from minutes to **<500ms**.
  • ▶ Enabled correlation of price volatility with news sentiment spikes.
Kafka Airflow TimescaleDB Docker Dash

BarçaBrain Analytics

Problem: Tactical football analysis requires manually sifting through vast, unstructured performance data, a slow and inefficient process.
Solution: Developed an AI-powered analytics tool using LangChain and a local LLM (Ollama) to create a conversational interface for complex football data. The system allows coaches and analysts to ask natural language questions about player performance, tactical formations, and match statistics.

  • ▶ Achieved **95% intent recognition** for complex tactical queries.
  • ▶ Reduced time-to-insight for match analysis by **70%**.
  • ▶ Integrated multiple data sources into a unified, queryable knowledge base.
